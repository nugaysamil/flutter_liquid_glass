## [0.0.1] - 2025-12-25

### Added
- Initial release of flutter_liquid_glass package
- `LiquidGlassCard` - Beautiful glass morphism card component
- `LiquidGlassContainer` - Flexible glass container with custom layer support
- `LiquidGlassBottomBar` - iOS-style bottom navigation bar with draggable indicator
- `LiquidGlassTabBar` - Horizontal tab bar with smooth animations and jelly physics
- `LiquidGlassAppBar` - Glass morphism app bar following Apple's design patterns
- `LiquidGlassSideBar` - Vertical navigation sidebar with optional header and footer
- `LiquidGlassToolBar` - Bottom toolbar for actions
- `LiquidGlassButton` - Interactive glass button with scale animation and glow effects
- `LiquidGlassTextField` - Glass morphism text input field
- `LiquidGlassLayer` - Global glass layer for grouped glass effects
- `LiquidGlassBlendGroup` - Seamless blending between glass surfaces
- Support for both grouped and standalone glass modes
- Customizable glass settings (thickness, blur, refractive index, etc.)
- Quality presets (standard and premium)
- Comprehensive example app demonstrating all components
