# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2025-12-25

### Added
- Initial release of flutter_liquid_glass package
- `LGCard` - Beautiful glass morphism card component
- `LGContainer` - Flexible glass container with custom layer support
- `LGBottomBar` - iOS-style bottom navigation bar with draggable indicator
- `LGTabBar` - Horizontal tab bar with smooth animations and jelly physics
- `LGAppBar` - Glass morphism app bar following Apple's design patterns
- `LGSideBar` - Vertical navigation sidebar with optional header and footer
- `LGToolBar` - Bottom toolbar for actions
- `LGButton` - Interactive glass button with scale animation and glow effects
- `LGTextField` - Glass morphism text input field
- `LiquidGlassLayer` - Global glass layer for grouped glass effects
- `LiquidGlassBlendGroup` - Seamless blending between glass surfaces
- Support for both grouped and standalone glass modes
- Customizable glass settings (thickness, blur, refractive index, etc.)
- Quality presets (standard and premium)
- Comprehensive example app demonstrating all components
